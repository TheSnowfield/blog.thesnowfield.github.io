我不會喜歡的幾種人

想了很多天, 終於還是寫下了這一篇文章。

畢竟在自己部落格嘛, 算是屬於自己的一片小天地, 想説什麽就説什麽了。
所以我會一點也不委婉的說: 與其説是不會喜歡, 不如直接一點, 就是令我非常討厭。
(當然大標題還是委婉一點吧… 嗯)

所以接下來我不會像在社交平台或者群内一樣非常委婉的去表達, 而是會用相對尖銳的語言去抨擊。
如果你明知道我可能不會喜歡你的話, 慎點。(另外, 請勿對號入座)

我是?

不管怎麽説, 就一般理性而言, 先不要着急去罵別人對不對。
我先自我檢討： “我, 是一個怎樣的人?”

我會給你說, 我是一個對是非觀念非常強烈的人。
我有自己的道德底綫, 也會去恪守。我知道什麽該做什麽不該做, 什麽事情該怎麽去做。

我對別人往往是包容的, 我會刻意去提醒自己要去兼容並包。
最大程度的理解和吸納別人的觀點, 對自己不一定是壞處, 從別的角度出發説不定會發現不一樣的新東西。

我重情義, 我會盡力去幫助身邊的人, 無論對方是誰。
需要我即是我存在的意義, 不需要我我隨時可以離開。

我對身邊人的好, 並不是爲了得到什麽, 而是我認爲這是我應該做的。
然而因爲對身邊的人太好, 常常會被誤會當成情感上的示好, 最後被誤解。在經歷過許多糟糕的事情之後, 這些歷史的傷痕使我不得不對你時刻保持一定距離和戒備心。

我是一個非常喜歡去思考, 並且富有感性但又不失理性的人。
我常常不在思考, 迸發出一些別人可能想不到的靈感。這些東西可能是一款新的設計, 也可能是對問題的全新解決方案, 當然也有可能是整活。我從來不缺靈感, 我缺的是實現靈感的時間和成本。

我會時刻盡可能保持低調, 低調的去做自己想做的事情, 去做自己喜歡的事情。
隱藏自己的實力, 會讓別人對你產生誤判, 這不是什麽壞事。始終保持神秘感, 這樣就能限制真正瞭解自己的人的範圍。
我覺得一個人的存在並不是爲了展示自己要多大多强, 樹大招風也正是如此。正所謂黑暗森林法則。

《葬送のフリーレン》中的女主(當然是芙莉蓮辣)的老師 弗蘭梅 説過:

“打敗魔族的方法是欺騙。故意去限制自己的魔力讓魔族誤判你的實力, 從而贏得勝利。爲什麽魔族不會這麽做? 因爲他們會把魔力當作階級的象徵, 誰的魔力越高誰就越强, 並且他們從骨子裡就做不到隱藏自己實力。”

這一段話與我產生了强烈的共鳴:
現在的人又何嘗不是這樣, 想有流量想變網紅, 不惜一切代價只爲博得別人的眼球。
在芸芸衆生中脫穎而出雖然也不見得是壞事, 但我覺得過度去張揚自己, 最終只會惹來災禍。

所以, 就此引出我討厭的第一種人。

第一種: 張揚的人

這類人非常好識別。因爲其性格張揚, 總是想著世界以他為中心。
至於嘴, 那就更是大喇叭了。不説還好, 一説就是在吹噓自己的能力如何強大, 如何如何厲害。

這類人我一般不會樂意去給多少面子。
我覺得這一類人就像《葬送のフリーレン》中的魔族, 無時無刻不在向周圍的人彰顯自己那可憐的不丁點的實力。我身邊很强的朋友手指頭都數不過來, 但是沒見過他們有一個是這樣的。

但是”市場”通常對這類人示好, 因爲”市場”只會提出需求, 而這類人張揚的行爲正好滿足了”市場”的需求。所以一般這類人可能會活得風生水起, 會被”市場”襯托變成所謂的大佬。

在大陸這種風氣十分常見, 小白圍著大佬轉: 大佬拉, 小白吃。大佬説, 小白信。
反正我是看不慣的了。倒是這種人漏洞很多, 拿去好好利用說不定可以減輕自己心智負擔 (

就此, 我例舉一下什麽地方常常出現這樣的人:

  • 機圈 (那種一言不合網絡下發脚本刪別人數據的)
  • CTF/ACM (當然很多人玩只是興趣愛好, 但是也完全不缺以炫技爲目的的。這也是我不喜歡此類活動的主要原因, 我也坦白我確實沒這方面的動力和腦子)
  • 音游 (自己打了很高的分就xjb曬(做法來看似乎沒什麽問題, 是的, 你打多少分跟我p關係沒有), 看到比自己分還高人的就跪拜(? 不是很懂此類迷惑社交行爲))

還有一些想不起來的, 以後想起來了補充。

第二種: 弱智/腦殘

此處的弱智和腦殘並不指生理的/醫學/先天方面的疾病, 我反而覺得這些人應該的到社會的更多幫助。

我指的是那種腦子裡只有批和色圖, 一天到晚閑的沒事幹, 在社交平台看誰不順眼就直接開罵的人。

個別整天駡人帶生殖器官, 可能是文化水平低了點, 不知道如何去委婉表達。
不論說什麽, 只要他看你不順眼或是你説的話跟他意見不合, 就會找各種茬來吐口水。理由更是千奇百怪, 邏輯也是亂七八糟。

“老子不管這麽多, 我就是看你不順眼, 我就要罵完才爽”

此類人常常活動在 貼吧/微博評論區/論壇/游戲 等地, 此類人一般都是喜歡呆在網絡的垃圾桶。
此類人常常被稱爲鍵盤俠, 我相信這類下頭人沒多少人會喜歡 (

第三種: 嘴閑的人/無腦起哄

何爲”嘴閑的人”? 不是上面的腦殘人種, 而是 “嘴真的很閑”。
這類人常常在群聊的時候, 故意 ot(Off Topic 岔開話題)來提升自己存在感。

場景一:

以上, C 的行爲就是典型的 ot。
他加入不了 A 和 B 之間的對話, 但是提出一個完全不相干並且基本是廢話去强行打斷聊天, 藉此來提升自己的存在感。
請注意: B 是在向 A 提出問題, 通常想儘快得到 A 的回復。但是 C 出來强行打斷話題, 把 A 的話題岔開了。C 的這個做法會讓 B 產生話題割裂感, 開始對 C 產生不滿。

如果還不懂, 那麽場景二:

你看, 這個時候 B 的 提問 “要我帶點攝影器材嗎” 被 C 完全打斷了。
A 因爲 C 的 ot, 把 B 暫時忘掉了, 導致 A 和 B 之間的溝通效率低下, 我估計明天 B 是沒心情去漫展了吧 (((

並且 C 出於 ot 的心態, 他是絕對沒有打算跟 A 去漫展的,
所以結論就是 C 的發言就是廢話, 沒有任何意義和營養。作用嘛…. 打斷聊天…嘴閑的

這樣的人, 我只能説身邊不少。
ot 不好, 會嚴重打斷別人的交流, 如果你沒話可説, 那我建議還是不要説。

還有那種真的衝著搞事來的, 那麽我們接著場景二繼續:

那就由不得 B 了, B 現在看 C 確實不爽, 但是爲了滿足 C 的請求, 還是允了。
我是覺得沒什麽毛病。

其實對這種人我還是比較偏向溫和的, 不頻繁的話我會直接忽略。
但是我提問因爲 ot 而遲遲得不到回復, 我會比較生氣。你可以想一下重要的事情被連續打斷是什麽感覺。

結語

非常感謝你能看到這裡。不知道你是否能理解我, 我真的為今天這幾類人的相處傷透了腦筋。
雖然討厭歸討厭, 但我一般還是會和正常人一樣去跟他們去溝通, 除非他們真的回了一個讓我高血壓的話…

偏向低調的人在這個鼓吹高調的社會中存活, 我個人還是覺得比較困難的。
畢竟相比之下會失去很多機會和可能性, 可能這就是不適應社會吧, 但是至少我問心無愧。

嗯.. 就這樣吧。
